# Issue tracker — WTMA Finance

**Type:** Local markdown (`.scratch/`)

Issues for this repo are tracked as markdown files under `.scratch/<feature>/` when not using GitHub Issues.

## Triage labels

| Role | Label |
|------|-------|
| Needs evaluation | `needs-triage` |
| Waiting on reporter | `needs-info` |
| Ready for agent | `ready-for-agent` |
| Ready for human | `ready-for-human` |
| Won't fix | `wontfix` |

## Domain docs

- `CONTEXT.md` — ubiquitous language (repo root)
- `docs/adr/` — architectural decision records
